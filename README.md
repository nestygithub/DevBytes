# DevBytes

DevByte Viewer is an example app that showcases using Room and a Repository to create an offline cache.
* Repository
* Room
* Retrofit
* Glide
* Work Manager
* Live Data
* MVVM Architecture
* Data Binding
* Navigation
* Binding Adapter
* Coroutines
